XEPMARKET-ALFA Demo Content
============================

This folder contains demo content for one-click import.

Files:
- demo-content.xml - Products, pages, and media (Export from WordPress > Tools > Export > All content)
- demo-widgets.wie - Widget settings (optional)

How to create demo-content.xml:
1. Go to WordPress Admin > Tools > Export
2. Select "All content"
3. Click "Download Export File"
4. Rename the file to "demo-content.xml"
5. Place it in this folder

The One Click Demo Import plugin will automatically detect and import this content.
